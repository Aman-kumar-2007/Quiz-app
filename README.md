# 🧠 Quiz App

A responsive and interactive Quiz Web Application built using **HTML, CSS, and JavaScript**.
This project allows users to test their knowledge through multiple-choice questions with a smooth and user-friendly interface.

---

## 🚀 Features

* 🎯 Dynamic question rendering
* ✅ Multiple choice answers
* 📊 Score tracking system
* 🔄 Next question navigation
* 🏁 Result display at the end
* 💻 Fully responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (ES6)** – Logic & interactivity

---

## 📂 Project Structure

```
quiz-app/
│── index.html
│── style.css
│── script.js
```

---

## 🎮 How It Works

1. User starts the quiz
2. Questions are displayed one by one
3. User selects an answer
4. Score updates based on correct answers
5. Final score is shown at the end
